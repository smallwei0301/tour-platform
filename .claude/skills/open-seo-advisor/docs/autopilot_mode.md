# 一鍵代操機器人（Autopilot）

給完全的新手：**一個指令，剩下交給它。** 不用懂 SEO、廣告、程式，也不用選
半天要用哪個模式。

```bash
seo-advisor auto https://你的網站.com
seo-advisor auto "幫我規劃餐飲來客成長方案"
seo-advisor auto-demo          # 免金鑰看範例
```

或直接執行 `seo-advisor` 進互動精靈，第一個選項就是「一鍵全自動」。

## 它會做什麼

1. **判斷你的目標** → 自動決定要出動哪些專家（SEO 顧問、CRO、UTM、電商、
   成效分析、內容、產圖…）。
2. **自動跑完所有分析**（免金鑰、用純邏輯或內建示範資料）。
3. **產出三份報告**：
   - `auto-report-beginner.md`：給你的白話懶人包（3 分鐘看懂最重要的事）。
   - `auto-report.md`：完整報告（可交給工程或行銷團隊）。
   - `cost-estimate.json`：會不會花錢的明細。

## 「放著不管」但不會誤燒錢：一次知情同意

這是這個工具最重要的安全設計。分析部分永遠免費、放著不管即可；但**會花錢、
寫入或對外發布的動作**，採「一次知情同意」：

1. 系統先把每個要花錢的動作整理成**白話明細**——例如「產 4 張圖大概多少費用」、
   「廣告每日預算最多增加多少錢」、「伺服器升級大概多少」，並標明風險與是否可回滾。
2. 你看懂後，**輸入一次同意確認字串**（例如 `APPROVE AUTO EXECUTION MAX TWD 500`）。
   單純打 `y` 不算同意——這是刻意的，避免手滑或金魚腦誤按就花錢。
3. 同意後，系統才自動執行**白名單內的安全動作**，你就可以放著不管。

不加 `--approve` 時，只會做免費分析，不執行任何花錢/寫入/發布動作。

## 同意也不會做的事（永遠不自動執行）

「同意」不是無限授權。以下動作**即使你同意了也永遠不會自動做**，一定停在計畫、
交給你或專業人士決定：

- 刪除資料 / 刪除廣告活動 / 刪除檔案
- 啟用新廣告投放、增加預算超過你授權的上限
- 付款、購買、升級付費方案
- 發送 Email / 簡訊 / LINE 給真實名單
- 對外發布新聞稿、社群貼文、正式上線內容
- 修改 DNS / SSL / 伺服器 / 資料庫
- 任何不可回滾的破壞性操作
- 法務 / 財務 / 醫療 / 投資建議的最終採用

## 目前版本（v0.1.7）

MVP 階段：分析全自動、免金鑰、3 分鐘出結果。為了確保絕不誤燒錢，**真實花錢
動作（產圖 API、LLM 產文、廣告預算調整）在本版一律停在「計畫」階段**，成本
明細會清楚呈現，實際執行留待後續版本並仍受上述安全邊界約束。這代表你現在就能
安心地「放著不管」拿到完整分析與方案，而不會有任何意外花費。
