## ▌互動狀態規格（全站適用）

| 狀態 | 樣式 |
|------|------|
| 按鈕 hover | 顏色加深 10%，`transition: 0.2s` |
| 按鈕 active（點擊中）| 顏色加深 20%，輕微縮小 `scale(0.98)` |
| 按鈕 disabled | `opacity: 0.4`，`cursor: not-allowed` |
| 輸入框 focus | `border: 2px solid #1B6B4A`，淡綠色光暈 |
| 輸入框 error | `border: 2px solid #E53E3E`，紅色提示文字 |
| 卡片 hover | `transform: translateY(-4px)`，陰影加深 |
| 收藏心型 hover | 填充紅色 `#E53E3E` |
| 連結 hover | 主色底線顯示 |
| 載入中 | 骨架屏（Skeleton），主色 pulse 動畫 |
| Toast 通知 | 右上角彈出，3秒後自動消失 |
