# 06. 金流計劃

## 6.1 台灣金流整合方案

### 推薦架構

```
消費者付款
    │
    ├── 信用卡 ──────────┐
    ├── ATM 轉帳 ────────┤
    ├── 超商代碼 ────────┤──→ ECPay 綠界（主要）
    └── LINE Pay ────────┘──→ LINE Pay API（Phase 3）
                              │
                              ▼
                    平台 ECPay 收款帳戶
                              │
                    活動完成後 T+7 結算
                              │
                    ┌─────────┴─────────┐
                    ▼                   ▼
             平台收入（15%）      導遊收款（85%）
                                   → 銀行轉帳
```

---

## 6.2 ECPay 綠界申請流程

### 申請條件
- 公司登記（有限公司或股份有限公司）
- 負責人身分證
- 公司存摺封面
- 網站 URL + 上線狀態

### 申請步驟
1. 前往 https://www.ecpay.com.tw 建立商家帳號
2. 填寫商家資料（公司名稱、統一編號、聯繫資訊）
3. 上傳審核文件
4. 審核時間：5~7 個工作天
5. 通過後取得 MerchantID + HashKey + HashIV
6. 測試環境先行串接（Stage）
7. 確認後切換正式環境（Production）

### ECPay 費率
| 付款方式 | 費率 |
|---------|------|
| 信用卡（一次付清） | 2.75% |
| 信用卡（分期） | 依期數 |
| ATM 轉帳 | NT$15/筆 |
| 超商代碼 | NT$25/筆 |
| LINE Pay | 2.5% |

### 月費
- 基本方案：NT$0（按交易計費）
- 信用卡月費：NT$500（若有信用卡功能）

---

## 6.3 LINE Pay 串接（Phase 3）

### 申請流程
1. 前往 LINE Pay 商家申請 https://pay.line.me
2. 需要公司登記 + 銀行帳號
3. 申請審核：7~14 個工作天
4. 取得 Channel ID + Channel Secret Key
5. 使用 LINE Pay Online API v3

### 適合場景
- 目標用戶 25~35 歲，LINE Pay 普及率高
- 結帳只需點開 LINE，轉換率比信用卡高 20%+

## 6.4 金流資安規範

### 絕對不做
- ❌ 不儲存任何信用卡號
- ❌ 不記錄 CVV / 有效期
- ❌ 不繞過金流商直接處理卡號

### 必須做
- ✅ 所有金流 Callback 使用 HMAC-SHA256 驗簽
- ✅ HTTPS only（Vercel 預設）
- ✅ 訂單狀態更新只接受 ECPay 官方 IP
- ✅ 交易 Log 保留 5 年（法規要求）

## 6.5 退款政策（請以 v2 主檔為準）

- 主檔：`docs/05-business/06-payment-plan/04-refund-policy-v2.md`（Source of Truth）

| 退款時機 | 退款比例 | 備註 |
|---------|---------|------|
| 出團前 7 天以上（168 小時以上） | 100% | 全額退款 |
| 出團前 72～168 小時 | 70% | 部分退款 |
| 出團前 72 小時內 | 0% | 原則上不退 |
| 導遊取消 / 平台取消 | 100% | 全額退款 |
| 不可抗力（限定類別） | 100% 或 1 次免費改期（二擇一） | `government_order`、`typhoon`、`earthquake`、`organizer_cancel` |

### 退款流程（作業摘要）
1. 旅客於 /orders/[id]/refund 提出申請
2. 系統依 v2 主檔與訂單時間判斷，進入退費或人工審核
3. 人工審核需保留證據並明確拒退/核准依據
4. 後台執行退款（ECPay 退款 API）
5. 退款入帳時效以作業目標與通路回報追蹤

> 備註：手續費由平台吸收，旅客不另收退款手續費。

## 6.6 導遊結算流程

### 結算時機
- 活動完成日 + 7 天（保留期，處理糾紛）
- 若有爭議，延長至 14 天

### 結算方式
1. 系統自動計算：訂單金額 × 85% - 轉帳手續費
2. 導遊提交提款申請（最低提款金額：NT$5,000）
3. 每週一、四批次轉帳（銀行轉帳）
4. 發送結算明細 Email

### 扣款項目
- 平台手續費：15%
- 銀行跨行轉帳費：NT$15/筆（同行免費）

---

## 6.7 財務對帳

### 每日自動對帳
- 系統撈取 ECPay 交易明細
- 比對內部訂單資料庫
- 差異 > NT$0 → 自動警報通知

### 月報告
- 每月 1 日自動生成上月報告
- 包含：GMV、平台收入、導遊結算、退款金額
- 自動發送給平台管理員
