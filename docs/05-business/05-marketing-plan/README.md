## 1. 品牌定位（重新確立）

### 一句話定位
> **「台灣最值得信任的在地導遊預約平台」**

### 對消費者說
> 找到真正了解台灣的導遊，不怕退款、不怕失聯、不怕行程變卦。

### 對導遊說
> 台灣抽成最低（15%），有自己的後台，接案從此不靠 LINE 群。

### 品牌名選項（競品參照後建議）
| 選項 | 意涵 | 英文友善度 |
|------|------|-----------|
| 島伴 | 台灣的島 + 旅伴 | 中 |
| 路引 (LuYin) | 引路人 + 在地路 | 高 |
| 導玩 | 導遊 × 玩樂 | 低 |
| TaiwanLocal | 直白、SEO 友善 | 高 |
| **WalkTaiwan** | 徒步在地，簡單好記 | ⭐ 推薦 |

## 2. 目標客群（細化）

### B2C — 消費者端

**主要客群 A：國內旅遊的台灣人（40%）**
- 年齡：25~45 歲
- 情境：週末帶家人出遊、想要有人規劃 + 帶路
- 痛點：不知道哪裡好玩、怕自己規劃麻煩
- 管道：Facebook、Instagram、LINE 群推薦

**主要客群 B：來台外國遊客（35%）**
- 年齡：20~40 歲，自由行為主
- 情境：無法中文，想找能溝通的在地人
- 痛點：台灣沒有好用的 Airbnb Experiences（Reddit 證實）
- 管道：Google 搜尋、Reddit r/taiwan、Instagram

**次要客群 C：企業團建 / 獎勵旅遊（25%）**
- 採購者：HR / 活動企劃
- 情境：20~50 人的員工旅遊，需要客製化規劃
- 管道：LinkedIn、業務開發、企業直接洽談

## 3. 獲客策略（競品分析後更新）

### Phase 1（0~6 個月）：佔領搜尋 + 建立信任

**SEO 策略（對抗 KKday/Klook 的核心武器）**
- KKday 和 Klook 的 SEO 主攻「熱門景點 + 活動商品」
- 我們的切入點：**長尾在地關鍵字**（KKday 不做的）

| 目標關鍵字類型 | 範例 | 月搜量估算 |
|--------------|------|-----------|
| 地區 × 活動 | 台東私人導遊推薦 | 中 |
| 情境型 | 花蓮一日遊帶路人 | 中 |
| 問題型 | 台灣自由行不會中文怎麼辦 | 高 |
| 競品否定型 | KKday 導遊推薦以外選擇 | 低但精準 |

**內容行銷（差異化 KKday 的制式商品描述）**
- 每個上架導遊：拍攝 30 秒自介影片（顯著差異化）
- 每條行程：導遊親筆的故事性描述（非制式格式）
- Blog：在地秘境系列（導遊自己寫，SEO 長尾流量）

**Reddit / 社群攻略（外國遊客）**
- 目標社群：r/taiwan、r/Taipei、r/solotravel、r/JapanTravelTips
- 策略：回答外國人「台灣哪裡好玩」的問題，自然帶入平台
- 注意：不要廣告式貼文，要真實推薦有說服力

---

### Phase 2（6~12 個月）：KOL 矩陣 + 企業客戶

**KOL 策略（對比 KKday 的廣撒網合作）**

KKday 找粉絲數大的 KOL 打知名度，我們不跟。
我們的策略：**微型 KOL（1萬~10萬粉）× 在地旅遊主題**

| KOL 類型 | 平台 | 合作方式 |
|---------|------|---------|
| 台灣各縣市旅遊帳號 | Instagram | 體驗置換 + 固定佣金 |
| 在地生活 Youtuber | YouTube | 導遊體驗影片 |
| 旅遊 Podcast | Podcast | 廣告 + 導遊訪談 |
| 外籍 Youtuber in Taiwan | YouTube | 英語市場開拓 |

**企業客戶直銷**
- 目標：100 人以下的公司 HR
- 管道：LinkedIn DM + 企業活動媒合平台
- 提案：客製化員工旅遊套餐，一站式搞定

---

### Phase 3（12 個月後）：口碑飛輪

**核心機制：讓導遊成為行銷員**
- 導遊每帶一個回頭客 → 額外抽成降至 12%（忠誠獎勵）
- 導遊推薦新導遊加入 → 獎勵金 NT$500
- 遊客推薦碼：推薦成功 → 下次預約折扣 NT$200

## 4. 品牌信任建立策略（KKday 最弱的點）

KKday Trustpilot 1.8/5 = 我們最大的行銷素材。

**「信任承諾」品牌宣言：**
1. 🔒 退款 3 天到帳（寫進首頁最顯眼位置）
2. ✅ 每位導遊實名驗證（徽章系統）
3. 📞 活動當天緊急熱線（30 分鐘回應）
4. 💰 最低抽成 15%（導遊自己會幫你宣傳）
