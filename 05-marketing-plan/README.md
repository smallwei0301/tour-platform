# 05. 行銷計劃

## 5.1 行銷策略總覽

**核心原則：** 先花時間，後花錢。先賣 100 個訂單，再談廣告。

### 兩條成長路徑

```
B2C（遊客獲取）: SEO → 社群 → KOL → 付費廣告
B2G（導遊獲取）: 直接聯繫 → 導遊公會 → 口碑
```

---

## 5.2 Phase 1：手動驗證期（Month 1~2）

**目標：** 不花廣告費，靠直接接觸獲得前 50 筆訂單

### 導遊獲取（供給側）
- [ ] 直接聯繫有人脈的 10 位導遊，請他們上架第一批活動
- [ ] 加入「台灣導遊協會」、「旅遊達人」Facebook 社群，觀察發文
- [ ] LinkedIn 尋找有導遊證的潛在合作者
- [ ] 提供早鳥導遊福利：前 50 位導遊手續費 10%（正式為 15%）

### 遊客獲取（需求側）
- [ ] PTT 旅遊版（Tourism）、背包客棧分享「找在地導遊」體驗文
- [ ] Facebook「台灣旅遊交流」類群組分享
- [ ] 直接聯繫 5~10 位旅遊 YouTuber / Instagramer 試用

---

## 5.3 Phase 2：內容行銷（Month 3~6）

**核心：** 建立「台灣在地旅遊」內容資產

### 三層內容策略

#### Level 1 — 教育型內容
- 「如何規劃台灣離島深度旅遊」
- 「找在地導遊 vs 跟團：差別在哪？」
- 「台灣各地區推薦行程整理」
- 發布平台：官方部落格（SEO 導流）

#### Level 2 — 啟發型內容
- 導遊故事系列：「這位導遊帶你看見不一樣的台東」
- 遊客分享系列：「用這個平台找到了我的夢中嚮導」
- 「Build in Public」：創辦人分享平台成長過程
- 發布平台：Instagram、LinkedIn

#### Level 3 — 娛樂型內容
- 台灣旅遊梗圖、短影片
- 導遊的一天 Vlog（YouTube Shorts / TikTok）
- 發布平台：Instagram Reels、TikTok、YouTube Shorts

### 內容行事曆

| 平台 | 頻率 | 內容類型 |
|------|------|---------|
| 官方部落格 | 每週 1 篇 | SEO 文章（教育型） |
| Instagram | 每週 3 則 | 混合（教育/啟發/娛樂） |
| Facebook 粉專 | 每週 2 則 | 活動推廣 + 內容分享 |
| YouTube | 每兩週 1 支 | 導遊故事 / 行程介紹 |
| Email Newsletter | 每月 1 封 | 精選行程 + 平台消息 |

## 5.4 SEO 策略

### 目標關鍵字（高搜尋量）
- 台灣旅遊行程
- 台灣導遊推薦
- 台東私人導覽
- 墾丁一日遊導遊
- 金門旅遊包車導遊

### 技術 SEO
- [ ] Next.js Metadata API（Open Graph / Twitter Card）
- [ ] 結構化資料（Schema.org：TouristAttraction、Event）
- [ ] Sitemap 自動生成
- [ ] 圖片 Alt Text + 延遲載入
- [ ] Core Web Vitals 優化（目標：LCP < 2.5s）

### 長尾策略：每個活動頁 = 一個 SEO 落地頁
- 標題格式：「[地點] [活動類型] — [導遊名稱] | 平台名稱」
- 例：「台東都蘭部落文化體驗 — 阿美族導遊 Wang | TourLink」

## 5.5 KOL / 網紅合作（Month 4~8）

### 目標 KOL 輪廓
- 粉絲數：1萬~10萬（微網紅，互動率高）
- 內容主題：台灣旅遊、生活、文化
- 合作形式：免費行程體驗 → 分享貼文 + 限動

### 合作方案
| 方案 | 條件 | 平台提供 |
|------|------|---------|
| 體驗合作 | 粉絲 5K+ | 免費行程（成本由平台補貼） |
| 聯名推廣 | 粉絲 2W+ | 分潤 5% + 免費行程 |
| 品牌大使 | 粉絲 10W+ | 月薪制 + 獨家代碼 |

## 5.6 付費廣告（Month 6+，有機成長後才啟動）

**原則：** 只在 CAC（獲客成本）< LTV（終身價值）時才投廣告

### 預估 LTV
- 平均訂單：NT$1,500 × 15% = NT$225 利潤
- 回購率假設：30%（同一遊客一年內再訂）
- LTV ≈ NT$225 × 1.3 = NT$292

### 廣告目標 CAC：< NT$150（LTV 的 50%）

### 廣告平台優先序
1. **Facebook / Instagram** — 旅遊受眾精準，支援台灣市場
2. **Google Search** — 關鍵字廣告（高意圖搜尋）
3. **LINE Ads** — 台灣用戶覆蓋率高
