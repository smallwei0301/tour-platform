# 09. 產品規格書

## 9.1 使用者故事（User Stories）

### 消費者（遊客）

```
AS A 自由行遊客
I WANT TO 瀏覽台灣在地活動清單
SO THAT 我可以找到符合我興趣的行程

AS A 自由行遊客
I WANT TO 查看導遊的背景和評價
SO THAT 我可以選擇可信任的導遊

AS A 自由行遊客
I WANT TO 選擇活動日期並線上付款
SO THAT 我可以確保預約成功

AS A 訂購者
I WANT TO 收到訂單確認通知
SO THAT 我確定行程已被接受

AS A 訂購者
I WANT TO 在活動結束後留下評價
SO THAT 我可以幫助其他遊客做決定
```

### 導遊

```
AS A 導遊
I WANT TO 申請加入平台並上架行程
SO THAT 我可以接觸到更多潛在客戶

AS A 導遊
I WANT TO 設定活動的可預約日期和名額
SO THAT 我可以控制接案量

AS A 導遊
I WANT TO 接收訂單通知並確認接受
SO THAT 我知道有新客人預約

AS A 導遊
I WANT TO 查看我的收益和提款
SO THAT 我可以管理我的收入
```

## 9.2 核心功能規格

### F01 — 活動列表頁
- 顯示所有已上架活動卡片
- 篩選條件：地區、活動類型、日期區間、價格區間、評分
- 排序：推薦 / 最新 / 評分 / 價格低到高
- 每頁 12 個卡片，支援無限滾動

**活動卡片顯示資訊：**
- 封面圖
- 活動標題
- 導遊名稱 + 頭像
- 地區標籤
- 評分（星等 + 評價數）
- 時長（小時）
- 每人價格

---

### F02 — 活動詳情頁
- 輪播圖（最多 8 張）
- 活動完整描述
- 行程細節（集合地點、包含 / 不包含項目、注意事項）
- 導遊簡介 + 評價預覽
- 日期選擇器（顯示可用 / 已滿 / 不可用）
- 人數選擇（1~最大定員）
- 價格計算（自動更新）
- 「立即預約」CTA 按鈕

---

### F03 — 預約流程
**Step 1：確認訂單**
- 活動名稱、日期、人數、總金額
- 聯絡資訊填寫（姓名、Email、手機）

**Step 2：選擇付款方式**
- 信用卡（ECPay）
- ATM 轉帳（ECPay）
- 超商代碼（ECPay）
- LINE Pay（Phase 3）

**Step 3：付款完成**
- 訂單建立成功頁
- 發送確認信 Email
- 訂單狀態：待確認 → 已確認（導遊確認後）

---

### F04 — 導遊個人頁
- 導遊照片 + 姓名
- 個人簡介
- 擅長領域標籤
- 導遊執照狀態（已驗證標章）
- 活動列表（此導遊上架的）
- 評價列表（最新 10 則）
- 平均評分

---

### F05 — 導遊儀表板
**Overview 頁：**
- 本月訂單數 / 收益 / 完成率
- 即將到來的行程（7 天內）
- 待處理訂單提醒

**活動管理：**
- 已上架活動列表
- 新增 / 編輯活動
- 設定可預約日期

**訂單管理：**
- 訂單列表（狀態篩選）
- 接受 / 拒絕待確認訂單
- 聯繫消費者

**收益頁：**
- 月收益圖表
- 訂單明細列表
- 提款申請

## 9.3 頁面地圖（Sitemap）

```
/ (首頁)
├── /activities (活動列表)
│   └── /activities/[id] (活動詳情)
├── /guides (導遊列表)
│   └── /guides/[id] (導遊個人頁)
├── /booking (預約流程)
│   └── /booking/[activityId]
├── /orders (我的訂單)
│   └── /orders/[id]
├── /auth (登入/註冊)
├── /guide
│   ├── /guide/apply (申請成為導遊)
│   └── /guide/dashboard (導遊後台)
│       ├── /guide/dashboard/activities
│       ├── /guide/dashboard/orders
│       └── /guide/dashboard/earnings
└── /admin (平台後台，僅管理員)
    ├── /admin/guides
    ├── /admin/orders
    └── /admin/reports
```
