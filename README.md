# Tour Platform — 台灣在地導遊接案平台

> **一句話定位：** 讓台灣在地導遊能直接接案、管理行程、收款，讓遊客能找到真正在地的旅遊體驗。

## 專案概覽

| 項目 | 內容 |
|------|------|
| 專案名稱 | Tour Platform（暫定，待品牌命名） |
| 定位 | 台灣在地導遊接案平台 |
| 差異化 | 導遊分發系統 + 在地化金流（ECPay/LINE Pay） |
| 目標市場 | 台灣本島 + 離島，鎖定 FIT 自由行旅客 |
| 商業模式 | 平台抽成 15%（每筆訂單） |
| 目標上線 | 2026 Q3 |
